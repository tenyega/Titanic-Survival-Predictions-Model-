# Titanic-Survival-Predictions-Model-
Titanic – Classification supervisée : préparation des données (imputation Age par titres, features Family_size/Alone), encodage catégoriel, entraînement et comparaison de plusieurs modèles (LogReg, Decision Tree, Random Forest, Gradient Boosting, Naive Bayes, SVM) avec évaluation (accuracy, matrice de confusion, precision/recall/F1).
